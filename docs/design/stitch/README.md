# Stitch 设计稿（参考模板）

本目录存放 [Google Stitch](https://stitch.withgoogle.com/) 导出的 UI 设计稿，是 `frontend/` Vue 应用的视觉与布局参考，**不是运行时前端**。

| 子目录 | 对应 Vue 页面 |
|--------|----------------|
| `chat_interface_with_agent_trace/` | `frontend/src/views/ChatView.vue` |
| `knowledge_base_upload_management/` | `frontend/src/views/UploadView.vue` |
| `session_history_dashboard/` | `frontend/src/views/SessionsView.vue` |
| `paper_rag_agent_logo/` | `frontend/public/logo.svg` |
| `paper_rag_agent/DESIGN.md` | 设计 token（颜色、字体、间距） |

实际产品界面请启动 `frontend/`（`npm run dev`）后访问 http://127.0.0.1:5173 。
